Welcome to RPG Limit Break! RPGLB is an annual event where gamers from around the globe gather to perform speedruns and support our efforts to raise money for NAMI, the National Alliance on Mental Illness, via a week-long, 24-hour livestream. RPG Limit Break is currently between events, but we sometimes host online events from the RPG speedrunning community, which you can watch below if we're currently live!
